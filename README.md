# 🛍️ eKart

eKart is a Flutter-based online shopping application designed to provide a simple and user-friendly shopping experience. Users can browse products, view product details, manage their shopping cart, and place orders through a clean and responsive interface.

---

## ✨ Features

- 🔐 User Authentication
- 🏠 Home Screen
- 🛍️ Product Listing
- 🔍 Product Search
- 📂 Product Categories
- 📄 Product Details
- ❤️ Wishlist
- 🛒 Shopping Cart
- ➕ Add & Remove Cart Items
- 💳 Checkout
- 👤 User Profile
- 📱 Responsive Design

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Flutter SDK
- Dart SDK
- Android Studio or VS Code
- Git
- Supabase Project

---

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/krish-solanki/eKart.git
```

### 2. Navigate to the project

```bash
cd eKart
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Configure Supabase

- Create a Supabase project.
- Copy your Project URL and Anon Key.
- Add them to your Flutter application configuration.

### 5. Run the application

```bash
flutter run
```
